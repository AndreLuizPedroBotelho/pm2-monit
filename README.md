### Clone

---

```sh
$ git clone https://github.com/AndreLuizPedroBotelho/pm2-monit.git
```
<br>

### Installation

---

```sh
$ cd pm2-monit

$ npm install
```
<br>

### Configure

---

Configure env file in backend folder using env.example as a base
<br>
<br>

### Usage

---

Start projects

```sh
$ npm start
```
<br>

### Tech Stack

- react 
- node
- typescript
- adonis
- lerna

<br>


### Required

---

- node >= 12.19.0 
- pm2
